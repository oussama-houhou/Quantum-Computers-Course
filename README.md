## 📋 Course Overview
This course explores the physical implementations of quantum computers.

### Key Topics Covered:
1.  **Overview of Technologies:** Two-level systems & DiVincenzo's Criteria.
2.  **Superconducting Qubits:** Josephson Junctions, Transmons, and Circuit QED.
3.  **Atomic Qubits:** Ion traps and atomic cooling.
4.  **Spin Qubits:** Silicon-based qubits and Diamond NV Centers.
5.  **Photonic Qubits:** Single photon sources and cluster states.
